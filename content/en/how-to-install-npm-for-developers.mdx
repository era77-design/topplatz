To install npm, you typically install Node.js, which bundles npm. This guide will walk you through the process for various operating systems, ensuring you have the Node Package Manager ready for your development projects.

## What You Need

- Computer with internet access
- Command Line Interface (Terminal/Command Prompt)

## Steps

### Step 1: Understand npm and Node.js Bundling

npm, the Node Package Manager, is the default package manager for Node.js. It's used to install, manage, and share packages (libraries, frameworks, tools) for JavaScript development. When you install Node.js, npm is automatically installed alongside it, making the setup process straightforward. This integrated approach ensures compatibility and ease of use for developers.

### Step 2: Download the Node.js Installer

Visit the official Node.js website (nodejs.org) to download the recommended installer for your operating system. The website typically offers two versions: an LTS (Long Term Support) version, which is stable and recommended for most users, and a Current version with the latest features. Choose the LTS version for stability unless you have a specific need for the latest features. For a more detailed look, you can also consult a [Comprehensive Step-by-Step Guide for Developers](/en/how-to-install-npm-step-by-step-guide).

### Step 3: Install Node.js on Windows

Once the Node.js `.msi` installer is downloaded, run it. Follow the prompts in the installation wizard, accepting the license agreement and clicking 'Next' through the default settings. Ensure that 'npm package manager' is selected for installation, which it is by default. Complete the installation, and Node.js along with npm will be set up on your Windows system.

### Step 4: Install Node.js on macOS

For macOS, open the downloaded `.pkg` installer and follow the on-screen instructions. The installer will guide you through the process of accepting the terms and choosing an installation location. Node.js and npm will be installed in standard system directories, making them accessible from your Terminal. Alternatively, you can use Homebrew with `brew install node` for a package manager-based installation.

### Step 5: Install Node.js on Linux (Ubuntu/Debian)

On Debian-based systems like Ubuntu, you can install Node.js and npm using your system's package manager. First, update your package list with `sudo apt update`. Then, install Node.js using `sudo apt install nodejs npm`. This command will fetch and install both Node.js and npm from the official repositories. For advanced users, using NodeSource repositories can provide newer versions.

### Step 6: Verify Your Installation

After the installation is complete, open your Command Prompt (Windows), Terminal (macOS), or shell (Linux). Type `node -v` and press Enter to check the Node.js version. Then, type `npm -v` and press Enter to check the npm version. If both commands return version numbers, your installation was successful and you're ready to start using npm.

### Step 7: Update npm (Optional but Recommended)

While npm is installed with Node.js, it's often updated more frequently. To ensure you have the latest stable version of npm, you can update it independently. Open your command line and run `npm install -g npm@latest`. The `-g` flag ensures npm is installed globally, allowing you to use it across all your projects. This keeps your package manager up-to-date with the newest features and bug fixes.


## Tips & Tricks

- Consider using Node Version Manager (NVM) to easily switch between different Node.js and npm versions for various projects.
- Always download installers from the official Node.js website (nodejs.org) to ensure security and authenticity.
- If you encounter permission errors on Linux or macOS, try running commands with `sudo` (e.g., `sudo npm install -g <package>`), but use it judiciously.

## Warnings

- ⚠️ Downloading Node.js or npm from unofficial sources can pose security risks to your system. Always use the official website or trusted package managers.
- ⚠️ Be cautious when globally installing packages (`npm install -g`). While convenient, it can sometimes lead to version conflicts across projects. Prefer local installations when possible.

## Frequently Asked Questions

**How do I install npm in Windows?**

To install npm in Windows, download the Node.js .msi installer from the official Node.js website. Run the installer and follow the on-screen prompts, accepting the default options. npm is bundled with Node.js and will be installed automatically.

**How do I install npm on Mac?**

On Mac, you can install npm by downloading and running the Node.js .pkg installer from nodejs.org. Alternatively, you can use Homebrew by opening your Terminal and running `brew install node`, which will install both Node.js and npm.

**How do I install npm on Ubuntu?**

To install npm on Ubuntu, open your terminal and update your package list with `sudo apt update`. Then, install Node.js and npm using `sudo apt install nodejs npm`. This command will fetch and set up both components from Ubuntu's repositories.

**What is npm and why is it important?**

npm (Node Package Manager) is the world's largest software registry and the default package manager for JavaScript's runtime Node.js. It's crucial for developers to manage dependencies, share reusable code, and streamline their development workflow for JavaScript projects.

**How do I verify if npm is successfully installed?**

After installation, open your command line interface (Terminal or Command Prompt) and type `npm -v`. If npm is installed correctly, it will display its version number. You can also check the Node.js version with `node -v`.

**Can I install npm separately from Node.js?**

While it's technically possible to install npm separately, it's highly recommended to install Node.js, as npm is bundled with it and designed to work seamlessly together. Installing Node.js ensures you have a compatible and stable environment for npm to function correctly.
