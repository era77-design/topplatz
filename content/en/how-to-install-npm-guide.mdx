To install npm, you first need to install Node.js, as npm is bundled with it. The simplest method involves downloading the official Node.js installer for your operating system and following the on-screen prompts, which automatically sets up both Node.js and npm.

## What You Need

- Computer with internet access
- Command line interface (Terminal/CMD/PowerShell)

## Steps

### Step 1: Check for Existing Installation

Open your command line interface (Terminal on macOS/Linux, Command Prompt or PowerShell on Windows). Type `node -v` and `npm -v` and press Enter after each. If versions are displayed, Node.js and npm are already installed. If you see "command not found" or similar, you need to proceed with the installation.

### Step 2: Download the Node.js Installer

Visit the official Node.js website (nodejs.org). You'll typically see two recommended versions: an LTS (Long Term Support) version and a Current version. For most users, the LTS version is recommended for stability. Download the installer package appropriate for your operating system (Windows, macOS).

### Step 3: Run the Node.js Installer

Locate the downloaded installer file (e.g., `.msi` for Windows, `.pkg` for macOS) and double-click to run it. Follow the on-screen prompts, accepting the license agreement and default installation settings. The installer will automatically include npm along with Node.js and configure your system's PATH variable.

> 💡 **Tip:** Ensure "Node.js runtime" and "npm package manager" components are selected during installation. For Linux users, package managers are often preferred, but the installer works too.

### Step 4: Verify the Installation

After the installer completes, open a **new** command line window (close any old ones first). Type `node -v` and `npm -v` again. You should now see the installed versions of Node.js and npm respectively. This confirms that both are successfully set up and accessible from your system's PATH.

> 💡 **Tip:** If you still get "command not found," try restarting your computer or checking your system's PATH environment variables. For a more comprehensive guide on installation across different OS, refer to [How to Install npm: A Step-by-Step Guide for All OS](/en/how-to-install-npm).

### Step 5: Update npm (Recommended)

While npm comes bundled with Node.js, it's often a slightly older version. To ensure you have the latest features and security updates, update npm itself. In your command line, run `npm install -g npm@latest`. The `-g` flag ensures it's installed globally, making the new version available system-wide.

### Step 6: Install Your First Package (Optional)

To confirm npm is fully functional, try installing a simple global package. A common choice is `create-react-app` or `nodemon`. For example, run `npm install -g create-react-app`. This command downloads and installs the package globally, making its command-line tools available for use.

> 💡 **Tip:** You can check installed global packages with `npm list -g --depth=0`.


## Tips & Tricks

- Always use the LTS (Long Term Support) version of Node.js for production environments due to its stability and long-term maintenance.
- Consider using a Node.js version manager like `nvm` (Node Version Manager) to easily switch between different Node.js and npm versions for various projects.
- If you encounter `EACCES` permission errors on macOS/Linux, avoid using `sudo` with npm. Instead, fix npm permissions or use a version manager like `nvm`.

## Warnings

- ⚠️ Ensure you download the Node.js installer only from the official website (nodejs.org) to avoid malware or compromised versions.

## Frequently Asked Questions

**How to install npm in Windows?**

On Windows, the simplest way to install npm is by downloading and running the official Node.js installer from nodejs.org. This installer includes npm and automatically configures it for your system. After installation, open a new Command Prompt or PowerShell and type `npm -v` to verify.

**How to install npm on Mac?**

For macOS, download the `.pkg` installer from the official Node.js website (nodejs.org). Run the installer and follow the prompts. Alternatively, you can use Homebrew by running `brew install node` in your terminal, which installs both Node.js and npm.

**How to install npm in Ubuntu?**

On Ubuntu and other Debian-based Linux distributions, you can install npm by first installing Node.js using your system's package manager. Recommended steps involve adding the NodeSource PPA, then running `sudo apt update` and `sudo apt install nodejs`. This command typically installs npm along with Node.js.

**What is npm and why do I need it?**

npm, or Node Package Manager, is the default package manager for Node.js. It's used to install, manage, and share JavaScript packages (libraries, frameworks, tools) for your projects. You need it to easily add external dependencies to your Node.js applications and front-end projects.

**How do I update npm to the latest version?**

To update npm to its latest version, open your command line interface and run the command `npm install -g npm@latest`. The `-g` flag ensures the update is applied globally across your system. After running the command, you can verify the new version with `npm -v`.

**What should I do if npm is not recognized after installation?**

If npm is not recognized, first ensure you've opened a *new* command line window after installation. If the issue persists, your system's PATH environment variable might not be correctly configured. You may need to manually add the Node.js installation directory to your PATH or restart your computer. For detailed troubleshooting, check [How to Install npm: Step-by-Step Guide for All OS](/en/how-to-install-npm-guide-all-os).
