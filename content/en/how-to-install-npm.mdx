To install npm, you first need to install Node.js, which bundles npm automatically. Download the appropriate Node.js installer for your operating system from the official Node.js website and follow the installation prompts. For a detailed walkthrough, refer to our [How to Install npm: A Complete Step-by-Step Guide](/en/how-to-install-npm). Once Node.js is installed, npm will be ready for use.

## What You Need

- Computer (Windows, macOS, or Linux)
- Internet connection
- Web browser
- Administrator/root privileges

## Steps

### Step 1: Check for Existing Node.js and npm

Before starting, it's good practice to check if Node.js and npm are already installed on your system. Open your terminal (macOS/Linux) or Command Prompt/PowerShell (Windows) and type `node -v` and then `npm -v`. If either command returns a version number, you have them installed. If not, or if the versions are outdated, proceed with the installation.

### Step 2: Download the Node.js Installer

Node.js comes bundled with npm, so installing Node.js automatically installs npm. Navigate to the official Node.js website (nodejs.org) in your web browser. You'll typically see two recommended versions: the LTS (Long Term Support) version and the Current version. For most users, the LTS version is more stable and recommended. Download the installer specific to your operating system (Windows, macOS, or Linux).

### Step 3: Run the Node.js Installation Wizard

Locate the downloaded installer file and double-click it to begin the installation process. Follow the prompts in the installation wizard. For most users, accepting the default installation settings is perfectly fine. The wizard will guide you through accepting the license agreement, choosing the installation directory, and selecting components to install, including npm. Click 'Install' and allow the process to complete.

### Step 4: Verify the Installation

After the installation wizard finishes, reopen your terminal or command prompt. It's important to open a new window to ensure the system's PATH variable is updated. Type `node -v` and press Enter, then type `npm -v` and press Enter. You should now see the version numbers for Node.js and npm displayed, confirming a successful installation.

### Step 5: Update npm to the Latest Version (Recommended)

While npm comes bundled with Node.js, it might not always be the very latest version. It's a good practice to update npm independently to ensure you have the most recent features and bug fixes. In your terminal or command prompt, run the command: `npm install -g npm@latest`. The `-g` flag ensures a global installation, meaning npm itself is updated system-wide.

### Step 6: Start Using npm

With npm successfully installed and updated, you're ready to start managing your JavaScript projects. You can now use npm to initialize new projects with `npm init`, install dependencies for existing projects using `npm install`, or add specific packages with `npm install <package-name>`. Explore the vast npm registry to find packages that can enhance your development workflow.


## Tips & Tricks

- Always use the LTS version of Node.js: For production environments and general development, the Long Term Support (LTS) version of Node.js is recommended due to its stability and extended support.
- Understand Global vs. Local Installs: Packages installed with the `-g` flag are global (system-wide) while those without are local (project-specific). Choose wisely based on your needs.
- Keep npm Updated: Regularly run `npm install -g npm@latest` to ensure your npm client is up-to-date, benefiting from the latest features and security patches. For more installation options, see our [How to Install npm: A Step-by-Step Guide for Beginners](/en/how-to-install-npm-guide).

## Warnings

- ⚠️ You may need administrator (Windows) or root (macOS/Linux with `sudo`) privileges to install Node.js and npm, especially for global installations. Always be cautious when granting such permissions.

## Frequently Asked Questions

**How to install npm in Windows?**

To install npm in Windows, download the Windows Installer (.msi) from the official Node.js website. Run the installer and follow the wizard's prompts, accepting the default settings. Node.js will automatically install npm alongside it.

**How to install npm on Mac?**

On a Mac, you can install npm by downloading the macOS Installer (.pkg) from nodejs.org. Run the package file and proceed through the installation steps. Alternatively, you can use Homebrew with `brew install node` in your terminal.

**How to install npm on Ubuntu?**

To install npm on Ubuntu, it's recommended to first install Node.js. You can do this by using the NodeSource repository for up-to-date versions: `curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -` followed by `sudo apt-get install -y nodejs`. This will install both Node.js and npm.

**Do I need Node.js to install npm?**

Yes, npm (Node Package Manager) is distributed with Node.js. When you install Node.js, npm is automatically installed as part of the package. You cannot install npm independently without first having Node.js on your system.

**What if 'npm -v' doesn't work after installation?**

If `npm -v` doesn't work, ensure you've opened a *new* terminal or command prompt window after installation. This refreshes the system's PATH variable. If it still fails, try restarting your computer or reinstalling Node.js to resolve potential path issues.

**What is the difference between global and local npm packages?**

Global npm packages are installed in a central location on your system and can be used by any project, often for command-line tools. Local packages are installed within a specific project's `node_modules` directory and are only available to that particular project, typically for project dependencies.
