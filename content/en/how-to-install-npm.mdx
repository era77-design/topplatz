Installing npm (Node Package Manager) is a fundamental step for any JavaScript developer, as it enables you to manage project dependencies and access a vast ecosystem of development tools. npm is automatically bundled with Node.js, making its installation straightforward across various operating systems. This guide will walk you through the process of getting npm up and running on your machine.

## What You Need

- Computer (Windows, macOS, or Linux)
- Internet connection
- Administrator privileges

## Steps

### Step 1: Understand Node.js and npm's Relationship

Before you begin, it's important to know that npm is distributed with Node.js. This means that when you install Node.js, npm is automatically installed alongside it. You do not need to install them separately. Node.js is a JavaScript runtime environment that allows you to run JavaScript code outside of a web browser.

> 💡 **Tip:** Always download Node.js from the official website (nodejs.org) to ensure you get the legitimate and most stable version.

### Step 2: Download the Node.js Installer

Navigate to the official Node.js website (nodejs.org) in your web browser. You will typically see two recommended versions: an LTS (Long Term Support) version and the Current version. For most users and production environments, the LTS version is recommended due to its stability and ongoing support. Download the installer appropriate for your operating system (Windows, macOS, or Linux).

> 💡 **Tip:** The website usually auto-detects your operating system and recommends the correct download. If not, select it manually.

### Step 3: Install Node.js (and npm) on Windows and macOS

Once the Node.js installer has downloaded, locate the file and run it. Follow the prompts in the installation wizard: accept the license agreement, choose the installation destination (default is usually fine), and select the components to install (ensure 'npm package manager' is checked, which it is by default). Complete the installation process. This will install both Node.js and npm on your system.

> 💡 **Tip:** It's often a good idea to restart your terminal or command prompt after installation to ensure that the system's PATH variable is updated correctly.

### Step 4: Install Node.js (and npm) on Ubuntu/Linux

For Ubuntu and other Debian-based Linux distributions, it's recommended to use NodeSource to get the latest LTS version of Node.js and npm. First, update your package list: `sudo apt update`. Then, install curl if you don't have it: `sudo apt install curl`. Next, add the NodeSource repository and install Node.js and npm: `curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -` followed by `sudo apt install nodejs`.

> 💡 **Tip:** Using a Node Version Manager (nvm) like `nvm install --lts` is another flexible way to manage multiple Node.js versions and their respective npm installations on Linux and macOS.

### Step 5: Verify Your Installation

After the installation is complete, open your terminal or command prompt and run the following commands to verify that both Node.js and npm have been installed successfully. Type `node -v` and press Enter to check the Node.js version. Then, type `npm -v` and press Enter to check the npm version. You should see version numbers displayed for both, confirming a successful installation.

> 💡 **Tip:** If you encounter 'command not found' errors, try restarting your computer or ensuring the Node.js installation directory is correctly added to your system's PATH environment variable. For more detailed troubleshooting, refer to [How to Install npm: A Step-by-Step Guide for All OS](/en/how-to-install-npm).

### Step 6: Update npm to the Latest Version (Optional but Recommended)

While npm comes bundled with Node.js, it's often updated independently. To ensure you have the absolute latest version of npm, you can update it globally using npm itself. Open your terminal or command prompt and run: `npm install -g npm@latest`. This command will update your npm client to the newest available version, providing you with the latest features and bug fixes.

> 💡 **Tip:** Regularly updating npm can prevent issues with package installations and ensure compatibility with newer project dependencies.

### Step 7: Basic npm Usage (Optional)

Now that npm is installed, you can start using it for your projects. To initialize a new project, navigate to your project directory in the terminal and run `npm init`. To install a package, use `npm install <package-name>`. For example, `npm install express` will install the Express.js framework locally in your project. These commands are fundamental for managing project dependencies.

> 💡 **Tip:** Always run `npm install` (without a package name) in a project directory that contains a `package.json` file to install all listed dependencies.


## Tips & Tricks

- Consider using a Node Version Manager (like nvm for macOS/Linux or nvm-windows) if you need to switch between different Node.js and npm versions for various projects.
- Always download Node.js from the official website (nodejs.org) to prevent security risks and ensure you get the correct files.
- If you encounter permission errors when installing global packages, you might need to fix npm's permissions or use `sudo` on Linux/macOS, though the latter is generally discouraged for global installs.

## Warnings

- ⚠️ Be cautious when running `npm install -g` as it installs packages globally, which can sometimes lead to version conflicts if not managed carefully.
- ⚠️ Never run `npm` commands as root (e.g., `sudo npm install`) unless absolutely necessary, as it can lead to permission issues and security vulnerabilities. Instead, fix npm's permissions if needed.

## Frequently Asked Questions

**How to install npm in Windows?**

To install npm in Windows, simply download the Node.js installer (.msi file) from the official Node.js website (nodejs.org) and run it. The installer automatically includes npm, so once Node.js is installed, npm will be ready for use.

**How to install npm on Mac?**

Installing npm on Mac is straightforward: visit the official Node.js website (nodejs.org), download the macOS installer (.pkg file), and follow the installation wizard. npm is bundled with Node.js, so it will be installed alongside it.

**How to install npm on Ubuntu?**

To install npm on Ubuntu, it's recommended to use the NodeSource repository for the latest stable Node.js versions. You'll add the repository using `curl` and then install Node.js (which includes npm) via `sudo apt install nodejs`. Alternatively, consider using nvm (Node Version Manager) for more flexibility.

**What is npm and why do I need it?**

npm (Node Package Manager) is the default package manager for Node.js, essential for JavaScript development. It allows you to manage project dependencies, install reusable code packages, and run various development tools, significantly streamlining your workflow.

**Do I need to install Node.js separately to get npm?**

No, you do not need to install Node.js and npm separately. npm is automatically bundled and installed as part of the Node.js installation package. Once Node.js is successfully installed, npm will be available on your system.

**How do I update npm to the latest version?**

To update npm to its latest version, open your terminal or command prompt and run the command `npm install -g npm@latest`. This command globally updates the npm client itself, independent of your Node.js version.

**What should I do if npm commands are not recognized after installation?**

If npm commands are not recognized, ensure that Node.js was installed correctly and that its installation directory is added to your system's PATH environment variable. Often, simply restarting your terminal, command prompt, or even your computer can resolve this issue by refreshing the environment variables.
