npm (Node Package Manager) is an essential tool for JavaScript development, allowing you to manage and share packages of reusable code. Installing npm typically involves installing Node.js, which bundles npm automatically. This guide will walk you through the process, ensuring you have npm ready for your projects.

## What You Need

- Computer with Internet Access
- Web Browser

## Steps

### Step 1: Understand npm and Node.js

npm (Node Package Manager) is the default package manager for Node.js, a JavaScript runtime environment. When you install Node.js, npm is typically installed alongside it, making it easy to manage project dependencies. It allows developers to share and reuse code packages, speeding up development. For a more detailed look at the basics, check out [How to Install npm: A Beginner's Step-by-Step Guide](/en/how-to-install-npm-beginner-guide).

### Step 2: Download the Node.js Installer

Visit the official Node.js website (nodejs.org). You'll see two recommended versions: an LTS (Long Term Support) version and a Current version. For most users, the LTS version is recommended as it's more stable and thoroughly tested. Download the installer appropriate for your operating system (Windows, macOS, or Linux).

### Step 3: Run the Node.js Installer

Locate the downloaded installer file and double-click to run it. Follow the prompts in the installation wizard. Accept the license agreement, choose the installation directory (the default is usually fine), and select the components to install. Ensure that 'npm package manager' is selected for installation. Complete the installation process.

### Step 4: Verify Node.js and npm Installation

Open your command prompt (Windows), Terminal (macOS), or a shell (Linux). Type `node -v` and press Enter. This should display the installed Node.js version. Next, type `npm -v` and press Enter. This will show your npm version. If both commands return version numbers, Node.js and npm are successfully installed.

> 💡 **Tip:** If you encounter 'command not found' errors, you might need to restart your terminal or computer, or ensure Node.js was added to your system's PATH environment variable.

### Step 5: Update npm (Optional but Recommended)

Although npm comes bundled with Node.js, it's often a good practice to update npm to its latest stable version independently. To do this, open your command line interface and run the command: `npm install -g npm@latest`. The `-g` flag ensures it's installed globally on your system.

> 💡 **Tip:** Keeping npm updated ensures you have access to the latest features, bug fixes, and security improvements for package management.

### Step 6: Start Using npm

With npm installed, you can now begin managing packages for your JavaScript projects. Navigate to your project directory in the terminal and run `npm init` to create a `package.json` file. Then, you can install packages using `npm install <package-name>`, or `npm i <package-name>`. `npm i` is a shorthand for `npm install`.


## Tips & Tricks

- Always prefer installing the LTS (Long Term Support) version of Node.js for stability in production environments.
- Understand the difference between global (`npm install -g`) and local (`npm install`) package installations. Global packages are available system-wide, while local packages are specific to your project.
- Regularly update npm to its latest version to benefit from new features, performance improvements, and security patches.

## Warnings

- ⚠️ Always download Node.js installers from the official nodejs.org website to prevent security risks from unofficial sources.

## Frequently Asked Questions

**How do I install npm in Windows?**

The easiest way to install npm on Windows is by downloading and running the official Node.js installer from nodejs.org. The installer automatically includes npm, and you simply follow the on-screen prompts to complete the setup.

**How do I install npm on Mac?**

For macOS users, installing npm involves downloading the Node.js `.pkg` installer from nodejs.org and running it. Alternatively, you can use a package manager like Homebrew with the command `brew install node` in your terminal.

**How do I install npm on Ubuntu?**

To install npm on Ubuntu or other Linux distributions, it's recommended to use your system's package manager. For Ubuntu, you can use `sudo apt update` followed by `sudo apt install nodejs npm`. Ensure you install both Node.js and npm.

**What is npm and why do I need it?**

npm (Node Package Manager) is a command-line tool that allows you to install, share, and manage JavaScript packages (libraries, frameworks, and tools). You need it to easily add external dependencies to your Node.js projects and manage project-specific packages efficiently.

**How do I check if npm is already installed?**

To check if npm is installed, open your terminal or command prompt and type `npm -v`. If npm is installed, it will display its version number. If not, you'll typically see a 'command not found' error, indicating you need to install it.

**Can I install npm without Node.js?**

While it's technically possible to install npm separately, it's highly discouraged and generally not necessary. npm is designed to work with Node.js, and the simplest and most reliable method is to install Node.js, which bundles npm automatically.
